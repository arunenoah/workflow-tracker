# /dashboard

Open the Workflow-Tracker Kanban board in your browser.

## What happens:
1. Starts Workflow-Tracker server (if not already running)
2. Automatically opens the dashboard in your default browser
3. Shows your Kanban board with all projects and tasks
4. Real-time WebSocket connection established

## Usage:
```
/dashboard
```

## Features available:
- ✅ View all projects
- ✅ Create and manage tasks
- ✅ Drag-and-drop tasks between stages
- ✅ See AI agent reviews
- ✅ Real-time collaboration with team
- ✅ Customize workflow stages

## Keyboard shortcuts:
- `Esc` - Close modals
- `↑ ↓` - Navigate tasks (in list view)
- `Enter` - Open selected task
